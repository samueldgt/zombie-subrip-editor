----- Zombie SubRip Editor v0.1 12 Feb 2016 -----

ABOUT ----------------------------------------

This Open Source software allows you to shift the entire time of a subrip subtitle file (extensi√≥n .srt) by passing to the program the path of yor .srt file and the time you want to shift.

Inputs:
1. A valid and existing subrip file's path with .srt extension
2. A valid numeric value of the time to shift in format: seconds.milliseconds

Outputs:
1. A valid subrip file with the shifted time, located at the same path of the input but with other name (it doesn't modify the original file by security reasons).

USAGE ----------------------------------------

1. You will need to have installed ruby on your system. 
2. Then download and get the enthire zombie subrip's files placing them in to a folder.
3. Open a terminal and move it to your program folder's location.
4. Run the main file "zombie_subrip_editor.rb" using the ruby command line sentence for run, like bellow:

ruby zombie_subrip_editor.rb

5. The program will run and will appear some instructions to use it, just read and follow the instructions, it's very easy.
